# 🚗 Auctioneers

**Auctioneers** is a simple React-based web project for browsing and managing car listings in an auction-like format. Users can filter, sort, and paginate through listings, mark cars as favourites, and view detailed pages for each vehicle.

Built with **React**, **React Router**, **Vite**, and styled using **Tailwind CSS**.

---

## ⚙️ Tech Stack

- 🧩 React
- 💨 Tailwind CSS
- 🌐 React Router
- ⚡ Vite
- 🧪 React Testing Library (some tests implemented)

---
